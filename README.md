# SD - Jogo da Forca - Cliente/Servidor

## Compilação

```bash
javac *.java
```

## Execução

### 1. Iniciar o servidor

```bash
java Server
```

### 2. Iniciar clientes (em terminais diferentes)

```bash
java Client
```

## Como Jogar

* O jogo começa automaticamente quando existem pelo menos 2 jogadores
* Cada ronda tem um tempo limite de 60 segundos
* Os jogadores devem selecionar se querem:

  * Adivinhar uma letra
  * Adivinhar a palavra completa
* e de seguida colocar a letraou palavra que querem
